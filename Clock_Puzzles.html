How to Play Clockwork Conundrum

The clock has numbers already placed on every node.

  • Click any node to start.
  • Then only click nodes that are exactly the previous number's value steps away (clockwise or counterclockwise).
  • Goal: Visit every node exactly once without getting stuck.
  • Use the 💡 Hint button if you're stuck.
